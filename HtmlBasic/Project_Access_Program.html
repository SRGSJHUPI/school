Reasoning builds a student's ability to think clearly, solve problems and make evidence-based decisions. Below are the main types, sample topics for each type, and a quick class-wise suggestion so you can drop this straight into your syllabus page.

4. Verbal & Non-Verbal Reasoning

Verbal focuses on words/sentences; non-verbal uses shapes, figures and spatial patterns.

  • Verbal: syllogism, blood relations, directions, analogies
  • Non-verbal: figure series, mirror/water images, paper folding, embedded figures

Class-wise Suggestion (Where to Start)

  • Class 3‚Äì4: Basic patterns, simple analogies, picture series, group games that teach reasoning.
  • Class 5‚Äì6: Number/letter series, basic syllogism ideas, simple puzzles, directional & relational problems.
  • Class 7‚Äì8: Syllogisms, blood relations, seating puzzles, classification, introductory data reasoning.
  • Class 9‚Äì10: Advanced analytical puzzles, critical reasoning, multi-step deductions, timed practice.
  • Class 11‚Äì12: Higher-order critical reasoning, data-sufficiency, olympiad/competitive level problems.

  • Weekly puzzle challenge ‚Äî rotate team leaders and publish winners.
  • Project: design a reasoning game (paper/digital) and present rules & strategy.
  • Assessments: short timed quizzes + one open puzzle per term for creativity scoring.

Why Include Reasoning?

It improves problem-solving, exam performance (competitive exams), computational thinking and helps students form evidence-driven opinions ‚Äî all core to Rising Nalanda's vision of scientific thinking.
